Landing Page Documentation
=======================

Functionality:
- Serves as the main entry point for the website
- Displays key information about the business/application
- May include call-to-action buttons

Code Structure:
- Main component in landing.js
- Uses React for rendering
- Styled with CSS modules

How to Run:
1. Ensure Node.js is installed
2. Run 'npm install' to install dependencies
3. Start development server with 'npm run dev'
4. Landing page will be available at the root URL
